Guide pour la génération d'images personnalisées

Suivez ces étapes pour générer et télécharger vos images personnalisées en utilisant notre API.

Étape 1: Entrez une description

Décrivez la personne que vous souhaitez générer en anglais. Par exemple, "a portrait of a woman with blonde hair and blue eyes"

Étape 2: Spécifiez le nombre d'images

Indiquez le nombre d'images que vous souhaitez générer.

Étape 3: Générez vos images

Cliquez sur le bouton "Générer" pour lancer le processus de génération d'images.

Étape 4: Téléchargement automatique

Les images sont automatiquement téléchargées sur votre ordinateur une fois générées.
